## FAQ

#### What are the Controlls?

- WASD - Move
- Left/Right Click - Attack/Interact
- F - Pick Up
- (shift +) Q - Drop (all)
- T - open command prompt


#### What Commands are available in-Game?

- An overview of the commands can be displayed when using the `help` command
